Required Education

To be a UI Engineer you need to have at least a bachelor's degree in Computer Science, Web Develpoment,
Human-Computer Interaction, Mobile Computing, Graphic Design, Software Engineering or a related field of study.

It is strongly encouraged for you to have a portfolio of your best work.

Skills that are looked for in a UI engineer include front-end web development, interactive media design, human-computer interaction,
usability testing, mobile development, and graphic design.
It is also important to have effectiver interpersonal communication skills such as, leadership and project management.

As a UI Engineer you need to have knowledge in devopement languages including: HTML, CSS, JavaScript, AJAX, JSON, jQuery, Java, Ruby on Rails, and SQL database.

It is also beneficially to have experience in knowing how to use programs such as Photoshop, Flash, Flex and Illustrator from the Adobe Creative Suite.
